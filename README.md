# 🕉️ Arunachala Guide
**Your Complete Tiruvannamalai Companion**

A clean, modern, and fully responsive single-page website designed to assist pilgrims and tourists visiting Tiruvannamalai. It provides essential information about the sacred Arunachaleswarar Temple, the Girivalam route, local accommodation, and emergency contacts.

## 📖 About The Project
Tiruvannamalai is a major spiritual hub, and visitors often need quick, reliable information. **Arunachala Guide** serves as a digital companion, offering a seamless user experience with a theme that contrasts beautifully with the spiritual essence of the temple (using deep greens, warm golds, and clean whites).

## ✨ Features
- **🏠 Hero Section:** Welcoming landing page with quick navigation.
- ** Girivalam Route Preview:** Details on the 14km sacred circumambulation, including distance, time required, and auspicious days.
- **🛕 Arunachaleswarar Temple:** Comprehensive details including Darshan timings, daily Pooja schedules, and temple history.
- **🏨 Accommodation Guide:** Categorized room options (Temple Choultries, Mid-Range, and Premium Hotels) with pricing and features.
- **🚨 Quick Links & Emergency:** Vital contact numbers for ambulance, police, railway, bus stand, and temple administration.
- **💬 WhatsApp Integration:** Floating WhatsApp button and direct "Connect with Us" links for instant communication.
- **📱 Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
- **🎨 Smooth Animations:** Scroll-triggered animations and smooth navigation.

## 🛠️ Tech Stack
- **HTML5** (Semantic structure)
- **CSS3** (Custom styling, CSS Variables, Flexbox, Grid)
- **JavaScript** (Vanilla JS for smooth scrolling and scroll animations)
- **Font Awesome** (For icons)
- **Google Fonts** (Playfair Display & Poppins)
